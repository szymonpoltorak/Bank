# Bank

Banking app with frontend and backend for Data Security in Information Technology Systems academic course.

## How to run it

Use the script `run-docker.sh` in `scripts` folder to run the docker container.

```shell
cd scripts
sh run-docker.sh
```

## Technology stack

1. Frontend

* Angular 16,
* TypeScript 5,
* Angular Material,
* Scss,
* Rxjs,
* Html.

2. Backend

* Java 17,
* Spring Boot 3.2.0,
* Spring Data JPA,
* Gradle Groovy,
* PostgreSQL,
* MapStruct,
* Jwt,
* Lombok.

3. Others

* Docker,
* Nginx.
